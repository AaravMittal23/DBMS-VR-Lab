SRM Institute of Science and Technology
Department of Computer Science
Pre-Test

Answer the following questions to test your knowledge before starting the experiment.

1. Which SQL clause is used to group rows that have the same values in specified columns?

2. Which aggregate function returns the total number of rows in a table?

3. What does the AVG() function return?

4. Which clause is used to filter groups after the GROUP BY clause has been applied?

5. What is the main difference between WHERE and HAVING clauses?

6. Which aggregate function can be used to find the highest value in a column?

7. In the query: SELECT department, COUNT(*) FROM employees GROUP BY department; What does COUNT(*) count?

8. Can aggregate functions be used in the WHERE clause?

9. What will be the result of: SELECT SUM(salary) FROM employees WHERE salary IS NULL;

10. Which of the following is a valid SQL query using GROUP BY?

11. What does the SUM() function do?

12. In a SELECT statement with GROUP BY, which columns can be selected without being in the GROUP BY clause?

13. What is the purpose of using aggregate functions with GROUP BY?

14. Which aggregate function ignores NULL values in its calculation?

15. What is the correct order of SQL clauses in a query with GROUP BY and HAVING?
