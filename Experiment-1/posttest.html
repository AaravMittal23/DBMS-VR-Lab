Experiment: Implementation of Aggregate Functions and Grouping Operations

SRM Institute of Science and Technology
Department of Computer Science
Post-Test

Answer the following questions to evaluate your understanding after completing the experiment.

1. Which clause is used to filter grouped data based on aggregate function results?

2. What will the query "SELECT COUNT(DISTINCT department) FROM employees;" return?

3. Which of the following queries will find departments with average salary greater than 50000?

4. What is the result of COUNT(*) when all rows have NULL values in a column?

5. Which aggregate function can be used with both numeric and non-numeric data types?

6. In a query with GROUP BY, can you use ORDER BY to sort the grouped results?

7. What does the query "SELECT department, MAX(salary), MIN(salary) FROM employees GROUP BY department;" return?

8. Can you use multiple columns in a GROUP BY clause?

9. What is the difference between COUNT(*) and COUNT(column_name)?

10. Which of the following is a valid use of aggregate functions?

11. What happens if you use an aggregate function without GROUP BY on a table with multiple rows?

12. In the query "SELECT department, AVG(salary) as avg_sal FROM employees GROUP BY department ORDER BY avg_sal DESC;", what does ORDER BY sort by?

13. Can you use aggregate functions in the SELECT clause without GROUP BY?

14. What is the purpose of using DISTINCT with COUNT()?

15. Which query correctly finds the total salary expenditure per department for departments with more than 10 employees?
