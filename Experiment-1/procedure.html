Experiment: Implementation of Aggregate Functions and Grouping Operations

SRM Institute of Science and Technology
Department of Computer Science
Procedure

Step-by-Step Procedure

Follow these steps to perform the experiment on Aggregate Functions and Grouping Operations.

  1. Create a Database Table
    Create a table named employees with the following structure:
    • id (INT, PRIMARY KEY)
    • name (VARCHAR)
    • department (VARCHAR)
    • salary (DECIMAL)
    • age (INT)
  2. Insert Sample Data
    Insert at least 10-15 sample records into the employees table with different departments (e.g., IT, HR, Finance, Sales) and varying salary values.
  3. Display All Records
    Use the SELECT statement to display all records from the employees table: 
    SELECT * FROM employees;
  4. Apply COUNT() Function
    Count the total number of employees: 
    SELECT COUNT(*) AS total_employees FROM employees;
    Count employees in each department: 
    SELECT department, COUNT(*) AS emp_count 
FROM employees 
GROUP BY department;
  5. Apply SUM() Function
    Calculate the total salary expenditure: 
    SELECT SUM(salary) AS total_salary FROM employees;
    Calculate total salary per department: 
    SELECT department, SUM(salary) AS dept_total 
FROM employees 
GROUP BY department;
  6. Apply AVG() Function
    Find the average salary: 
    SELECT AVG(salary) AS avg_salary FROM employees;
    Find average salary per department: 
    SELECT department, AVG(salary) AS avg_salary 
FROM employees 
GROUP BY department;
  7. Apply MIN() and MAX() Functions
    Find the minimum and maximum salary: 
    SELECT MIN(salary) AS min_salary, MAX(salary) AS max_salary 
FROM employees;
    Find min and max salary per department: 
    SELECT department, MIN(salary) AS min_sal, MAX(salary) AS max_sal 
FROM employees 
GROUP BY department;
  8. Use GROUP BY with Multiple Columns
    Group by department and find statistics: 
    SELECT department, 
       COUNT(*) AS emp_count,
       AVG(salary) AS avg_salary,
       SUM(salary) AS total_salary
FROM employees 
GROUP BY department;
  9. Apply HAVING Clause
    Find departments with more than 3 employees: 
    SELECT department, COUNT(*) AS emp_count 
FROM employees 
GROUP BY department 
HAVING COUNT(*) > 3;
    Find departments with average salary greater than 50000: 
    SELECT department, AVG(salary) AS avg_salary 
FROM employees 
GROUP BY department 
HAVING AVG(salary) > 50000;
  10. Combine WHERE and HAVING
    Filter employees by age first, then group and filter groups: 
    SELECT department, AVG(salary) AS avg_salary 
FROM employees 
WHERE age > 25 
GROUP BY department 
HAVING AVG(salary) > 40000;
  11. Use ORDER BY with GROUP BY
    Sort the grouped results: 
    SELECT department, AVG(salary) AS avg_salary 
FROM employees 
GROUP BY department 
ORDER BY avg_salary DESC;
  12. Observe and Analyze Results
    Compare the results of different queries and understand how aggregate functions work with grouping operations. Note the difference between WHERE and HAVING clauses.

Expected Outcomes

  • Understanding of how aggregate functions summarize data
  • Knowledge of GROUP BY clause for grouping rows
  • Understanding the difference between WHERE and HAVING clauses
  • Ability to write complex queries combining aggregate functions, GROUP BY, HAVING, and ORDER BY
