Experiment: Implementation of Aggregate Functions and Grouping Operations

SRM Institute of Science and Technology
Department of Computer Science
Aim

Objective

The aim of this experiment is to understand and implement SQL aggregate functions such as COUNT, SUM, AVG, MIN, and MAX, and to apply grouping operations using GROUP BY and HAVING clauses to generate summarized reports from relational databases.

Learning Objectives

  • To understand the concept and usage of SQL aggregate functions
  • To learn how to use COUNT, SUM, AVG, MIN, and MAX functions effectively
  • To implement GROUP BY clause for grouping rows based on column values
  • To apply HAVING clause to filter grouped results
  • To differentiate between WHERE and HAVING clauses
  • To write complex SQL queries combining aggregate functions with grouping operations
  • To analyze and summarize data from relational databases

Expected Outcomes

Upon completion of this experiment, students will be able to:

  • Write SQL queries using various aggregate functions
  • Group data using GROUP BY clause
  • Filter grouped data using HAVING clause
  • Generate meaningful reports and summaries from database tables
  • Understand the practical applications of aggregate functions in real-world scenarios
